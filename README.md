# Java Multi-threaded TCP Server 

Esempi di server TCP multi-thread in Java che dimostrano diverse funzionalità di elaborazione dati e gestione delle connessioni client-server.

## Descrizione

Implementazioni di server TCP che gestiscono richieste client in modo concorrente usando thread separati. Ogni esempio dimostra pattern diversi di elaborazione dati e gestione delle eccezioni robusta.

##  Caratteristiche 

- **Architettura multi-thread**: Ogni client viene gestito da un thread dedicato
- **Gestione robusta delle eccezioni**: Try-catch a più livelli per parsing ed I/O
- **Chiusura sicura delle risorse**: Blocchi finally con controlli null e gestione separata
- **Pattern comando-risposta**: Protocolli testuali per comunicazione client-server
